# 수의병리학2 기말 인터랙티브 퀴즈

브라우저에서 클릭하며 풀고 즉시 채점·해설 보는 자가테스트.

## 여는 법

**로컬(오프라인, 가장 간단):**
- `quiz/index.html` 더블클릭 → 브라우저에서 바로 실행. 인터넷·서버 불필요.
- 노트북·데스크탑 어디서든 `git pull` 후 열면 됨.

**휴대폰/GitHub Pages(선택):**
- repo가 public이면 Settings → Pages → Branch `main`/`(root)` 켜면
  `https://<유저명>.github.io/study-vet/quiz/` 로 폰에서도 풀 수 있음.
- ⚠️ Pages는 저장소를 공개로 노출하므로 강의자료 비공개 원하면 로컬 사용 권장.

## 구성 (54문항)

- **심혈관 23 · 비뇨기 16 · 내분비 15**
- **족보 원본 9문항**(실제 기출, Claude 독립판정 정답 + 족보 원본정답 병기)
- **모두 고르시오 9문항**(정답 개수 비공개 — 실제 시험 포맷 모사)
- **사진문제 5문항**(강의안 figure 판독)

## 기능

- 필터 칩: 전체 / 과목별 / 족보 / 사진 / 모두고르시오 / **오답**(틀린 것만 재시험)
- 선지 클릭 → "정답 확인" → 정답(초록)·오답(빨강)·놓친정답(노랑) 표시 + 해설
- 점수·진행 자동 집계, 새로고침해도 유지(localStorage)
- 🔀 섞기, ↺ 초기화, 키보드 ←/→ 이동, 사진 탭하면 확대

## 주의

- 정답·해설은 **제공된 강의안·개념학습 자료 기준**. 족보 원본 정답이 틀린 경우 강의안 근거로 재판정함.
- "모두 고르시오"는 답이 1개일 수도, 2개 이상일 수도 있음(실제 시험과 동일하게 개수 안 알려줌).
